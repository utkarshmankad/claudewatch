Welcome to ClaudeWatch

Session usage monitoring for Claude.ai — three quick steps to get started.

1

Open claude.ai

ClaudeWatch reads your session usage directly from the claude.ai page. Open it in a tab and the extension will start capturing data immediately — no API key or login required.

Open claude.ai →
claude.ai
Session usage
2

Start ClaudeWatch Core

The Core daemon (part of the ClaudeWatch CLI) runs locally and stores your usage history, fetches detailed cost data from the Anthropic Admin API, and sends email/desktop alerts when spend thresholds are crossed.

$ claudewatch start

Core listens on http://localhost:7734 by default. You can change this in .

Terminal
$ claudewatch start
✓ Core listening on :7734
Polling Anthropic API…
3

You're set

ClaudeWatch will notify you as your session usage climbs. Click the extension icon any time to see live usage. Alerts fire automatically at:

  • 80% Approaching limit — wrap up your work
  • 90% High usage — save everything
  • 95% Critical — session almost exhausted

All thresholds are configurable in .

CW
ClaudeWatch — Session Alert Session 90% used (90,000 / 100,000 tokens)

🔒 Privacy: All data stays on your device. ClaudeWatch sends nothing to external servers — usage data goes only to the Anthropic API (to fetch your own account data) and optionally to Core running at localhost.
